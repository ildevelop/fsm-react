### Installation

1. Clone the repository:

   ```bash
   git clone https://github.com/ildevelop/fsm-react.git
   cd fsm-react

   ```

2. Install dependencies:

```bash
  yarn
```

3. Running the Application:

```bash
  yarn dev
```
